使用 [vuetifyjs](https://vuetifyjs.com) 和 [nuxtjs](https://nuxtjs.org) 构成的简易前端

可以输入用户名以及密码 然后选中用户 然后做题
所以是很简易的前端了 ~~没有什么特色功能~~

服务端包括了一些我个人抓取的 private api  而且写得很烂 暂时不适合开源
## 生成版本号
    node genbuild.js GIT_COMMIT_HASH
## 调试
    yarn dev

## 打包

    yarn generate
